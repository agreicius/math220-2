Theorem 1.4.5. Integrable functions.

Let \(f\) be a function defined on the interval \(I=[a,b]\text{.}\)
  1. If \(f\) is continuous on \(I\text{,}\) then \(f\) is integrable on \(I\text{.}\)
  2. If \(f\) is continuous everywhere on \(I\) except for a finite set of jump discontinuities, then \(f\) is integrable on \(I\text{.}\)
