Let \(f\) be a nonnegative function defined on the interval \([a,b]\text{,}\) and let \(\mathcal{C}\) be the graph of \(f\) from \(x=a\) to \(x=b\text{.}\) To estimate the area between \(\mathcal{C}\) and the \(x\)-axis proceed as follows:
  1. Divide \([a,b]\) into \(n\) equal subintervals, each of width \(\Delta x=(b-a)/n\text{.}\)
  2. For each subinterval pick a sample input \(x^*\) in that interval and build the rectangle whose base is the subinterval and whose sample height is given by \(h=f(x^*)\text{.}\) The area of this block is
    \begin{equation} A=\underset{\text{ height } }{\underbrace{f(x^*)}}\,\underset{\text{ width } }{\underbrace{\Delta x}}\text{.}\tag{1.2.1} \end{equation}
  3. Sum together the areas of the \(n\) blocks constructed in Step 2.
