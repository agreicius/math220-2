The function \(\arcsin\) is an increasing function with domain \([-1,1]\) and range \([0,\pi]\text{.}\) It satisfies the following properties:
\begin{align*} \arcsin(x)=\theta \amp \iff \sin\theta=x \text{ and } -\pi/2\leq \theta\leq \pi/2\\ \arcsin(\sin \theta)\amp =\theta \text{ for all } -\pi/2\leq \theta\leq \pi/2\\ \sin(\arcsin x)\amp =x \text{ for all } -1\leq x\leq 1\text{.} \end{align*}
