Item.

The function \(\arccos\) is a decreasing function with domain \([-1,1]\) and range \([0,\pi]\text{.}\) It satisfies the following properties:
\begin{align*} \arccos(x)=\theta \amp \iff \cos\theta=x \text{ and } 0\leq \theta\leq \pi\\ \arccos(\cos \theta)\amp =\theta \text{ for all } 0\leq \theta\leq \pi\\ \cos(\arccos x)\amp =x \text{ for all } -1\leq x\leq 1\text{.} \end{align*}
