Let \(f\) and \(g\) be integrable over \([a,b]\text{.}\)
  1. \(\int_a^a f(x)\, dx=0\text{.}\)
  2. Order of integration.
    \(\int_b^af(x)\, dx=-\int_a^b f(x)\, dx\text{.}\)
  3. Sum and difference.
    \(\displaystyle \int_a^b f(x)\pm g(x)\, dx=\int_a^bf(x)\, dx\pm \int_a^bg(x)\, dx\)
  4. Scalar multiple.
    \(\int_a^bcf(x)\, dx=c\int_a^bf(x)\, dx\) for any \(c\in \R\text{.}\)
  5. Interval partition.
    For any \(c\in \R\) we have
    \begin{equation*} \int_a^bf(x)\, dx=\int_a^cf(x)\, dx+\int_c^b f(x)\, dx\text{.} \end{equation*}
  6. Min-max inequality.
    If \(f\) satisfies \(m\leq f(x)\leq M\) for all \(x\in [a,b]\text{,}\) then
    \begin{equation*} m(b-a)\leq \int_a^b f(x)\, dx\leq M(b-a) \end{equation*}
  7. Domination.
    If \(f(x)\leq g(x)\) for all \(x\in [a,b]\text{,}\) then
    \begin{equation*} \int_a^bf(x)\, dx\leq \int_a^bg(x)\, dx\text{.} \end{equation*}
